# CARAVAN Search Engine

The search engine part of [CARAVAN](https://github.com/crest-cassia/caravan) framework.

## Prerequisite

Following versions of python 2 and 3 are supported.

- Python 2.7 or later
- Python 3.2 or later

## Running tests

```
python -m unittest discover test
```
